# Implementationsplan (Arkiverad)

Denna plan användes för den initiala uppsättningen av projektet.

## Vad som gjordes
1. **Agent Procedur**: `.aidocs/agent_procedure.md` skapades med strikta riktlinjer för agenter (AI).
2. **Projektstruktur**: Mapparna `frontend/`, `backend/` och `docs/` skapades.
3. **Git & GitHub**: Projektet initierades med git och pushades till GitHub under `uppfinnarnoa-lab/tag-across-sthlm`.

## Etablerad Procedur
Inför varje framtida kodändring måste följande procedur tillämpas:
1. Skapa `plan.md`
2. Implementera ändringar
3. Utför Audit
4. Arkivera plan
5. Generera Deploy-kommando
6. Uppdatera Dokumentation
7. Git Push
